About Me

My name is Michael Vreeken, I've been a developer/coder of many sorts over the past 15 years. I've explored numerous languages, including C++, Actionscript, Python and others, but have happily settled into web and android development.

Within the realm of web development I mostly stick to PHP Laravel for back end, and Bootstrap, jQuery and/or Vue.js for front end. I've used Node.js, Angular, and React with some success, but, personally, I find the most joy and productivity when sticking to Laravel and Vue.js.

More recently I've jumped into Android App development using java. Luckily this allows me to continue to use my back end programming talent for RESTful services and combine that with the convenience and user experience of a native app.

On the off chance that I'm not tinkering with some code you'll probably find me playing a video game or saving the world through Dungeons & Dragons.
